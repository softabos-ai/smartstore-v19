# 🎉 SmartStore - متجر التطبيقات الذكي

## 🚀 نظرة عامة

**SmartStore** هو تطبيق متجر ذكي مستقل يسمح لك بتضمين أي APK داخل مجلد `assets` وعرضه بواجهة احترافية شبيهة بمتجر Google Play.

---

## ✨ المميزات

### 🎯 **المميزات الرئيسية**:

✅ **قراءة APK تلقائياً** من مجلد assets  
✅ **عرض معلومات التطبيق** (الاسم، الأيقونة، الإصدار، الحجم)  
✅ **واجهة احترافية** شبيهة بمتجر Google Play  
✅ **تثبيت صامت** بدون أخطاء  
✅ **فتح تلقائي** بعد التثبيت  
✅ **ذكاء التحقق**: إذا مثبت → يفتحه، إذا غير مثبت → يثبته  
✅ **دعم Android 5.0 - 14+**

## ⚡ البدء السريع

### **الطريقة 1: استخدام APK الجاهز**

```bash
# 1. تثبيت التطبيق على الهاتف
adb install SmartStore-Installer.apk

# 2. فتح التطبيق
# سيعرض معلومات Rainbow V16 (المضمن افتراضياً)

# 3. الضغط على "تثبيت"
# سيثبت Rainbow ويفتحه تلقائياً!
```

### **الطريقة 2: تضمين APK الخاص بك**

```bash
# 1. احذف APK القديم
rm SmartStore/app/src/main/assets/embedded_app.apk

# 2. ضع APK جديد
cp /path/to/your-app.apk SmartStore/app/src/main/assets/embedded_app.apk

# 3. أعد بناء التطبيق
cd SmartStore
./gradlew clean assembleDebug

# 4. ستجد APK الجديد في:
# app/build/outputs/apk/debug/app-debug.apk
```

## 🎯 كيفية العمل

### **1. فتح SmartStore**:
- يقرأ APK من assets تلقائياً
- يستخرج معلومات التطبيق (الاسم، الأيقونة، الإصدار، الحجم)
- يعرضها في واجهة احترافية

### **2. الضغط على "تثبيت"**:
- يطلب صلاحية التثبيت (إذا لم تكن ممنوحة)
- يثبت التطبيق المضمن
- يفتح التطبيق تلقائياً بعد التثبيت

### **3. المرات القادمة**:
- يتحقق من حالة التطبيق
- إذا مثبت: يعرض زر "فتح التطبيق"
- إذا غير مثبت: يعرض زر "تثبيت"

## 🔧 التخصيص

### **تغيير APK المضمن**:

```bash
# الخطوة 1: ضع APK جديد
cp your-app.apk SmartStore/app/src/main/assets/embedded_app.apk

# الخطوة 2: أعد البناء
cd SmartStore
./gradlew clean assembleDebug
```

⚠️ **مهم**: اسم الملف يجب أن يكون `embedded_app.apk` بالضبط!

### **تغيير الألوان**:

عدّل `SmartStore/app/src/main/res/values/colors.xml`:

```xml
<color name="primary_color">#01875f</color>     <!-- اللون الأساسي -->
<color name="accent_color">#00d97e</color>       <!-- اللون الثانوي -->
```

```java
txtRating.setText("4.9");           // التقييم
txtDownloads.setText("50K+");       // عدد التنزيلات
```

## 🐛 حل المشاكل

### **مشكلة 1: "لم يتم العثور على APK"**

**السبب**: اسم الملف خاطئ أو غير موجود

**الحل**:
```bash
# تحقق من وجود الملف
ls -la SmartStore/app/src/main/assets/

# يجب أن يكون الاسم: embedded_app.apk
mv your_app.apk embedded_app.apk
```

### **مشكلة 2: "فشل التثبيت"**

**السبب**: صلاحية التثبيت غير ممنوحة

**الحل**:
1. افتح **الإعدادات** → **التطبيقات** → **SmartStore**
2. **الأذونات** → فعّل **"تثبيت تطبيقات غير معروفة"**

### **مشكلة 3: "التطبيق المضمن لا يفتح"**

**السبب**: التطبيق المضمن لا يحتوي على Activity رئيسية

**الحل**: تأكد من أن APK يحتوي على `LAUNCHER` intent filter
